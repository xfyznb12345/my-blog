# 1 从输入一个 URL 地址到浏览器完成渲染的整个过程

1.浏览器地址栏输入`URL`并回车,浏览器查找当前URL是否存在缓存，并比较缓存是否过期。
2. DNS解析URL得到对应IP
3. 发起TCP连接（三次握手）
4.  4. 发送http请求
5. 服务器处理请求，浏览器接受 HTTP 响应
6. 浏览器解析并渲染页面
7. 关闭TCP连接（四次挥手）
